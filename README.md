## Running Locally

No install needed — just open the file in a browser:

```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
# Open index.html in your browser
```

Or with VS Code's Live Server extension:
1. Open the folder in VS Code
2. Right-click `index.html` → **Open with Live Server**

---

## Deployment

Hosted on **GitHub Pages**. Any push to the `main` branch automatically updates the live site.

To enable GitHub Pages on your own fork:
1. Go to **Settings → Pages**
2. Set source to `main` branch, `/ (root)` folder
3. Save — your site will be live at `https://yourusername.github.io/portfolio/`
